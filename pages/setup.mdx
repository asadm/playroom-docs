## Using npm
Install Playroom Kit and its peer dependencies:

```sh npm2yarn copy
npm i playroomkit react react-dom
```


Then import in your project:
```js copy
import { onPlayerJoin, insertCoin, isHost, myPlayer } from "playroomkit";
```
## Using a CDN

Include via script tag (e.g. `unpkg.com` or `cdn.joinplayroom.com`). Include peer dependencies (react, react-dom) as needed:

```html copy
<body>
  <script src="https://unpkg.com/playroomkit/multiplayer.full.umd.js" crossorigin="anonymous"></script>
  <script>
    // Playroom Kit is available as global Playroom Kit
    Playroom.doSomething()
  </script>
</body>
```

Use the global `Playroom` scope in your app.
