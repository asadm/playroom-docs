# API Reference

Playroom Kit provides APIs for JavaScript/TypeScript, React, and Unity (C#).

## JavaScript / TypeScript

The core `playroomkit` package exposes functions for room management, player state, matchmaking, RPC, and more.

Browse the full [JavaScript API reference](/api-reference/js).

## React Hooks

Playroom Kit ships React hooks for use in React and React-based frameworks.

Browse the full [React Hooks reference](/api-reference/react).

## Unity (C#)

The Unity SDK exposes a C# API for use in Unity games.

Browse the full [Unity API reference](/api-reference/unity).
