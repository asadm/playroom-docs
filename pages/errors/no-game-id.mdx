# No gameId provided in insertCoin

You called `insertCoin()` without a `gameId`. This will impact analytics and enforce DAU limits soon.

```
insertCoin() called without gameId. This may impact analytics and enforce DAU limits soon.
Fix it here: https://docs.joinplayroom.com/errors/no-game-id
```

To fix it, create a project on the [Playroom Dev Portal](https://dev.joinplayroom.com) and paste your `gameId` into `insertCoin`.

## How to get your Game ID

<Steps>

### Go to the Dev Portal

Navigate to [dev.joinplayroom.com/signin](https://dev.joinplayroom.com/signin). You'll see the **Dev mode** sign-in form.



### Enter your email

Type your email address and click **Continue**.

![Playroom Dev Portal signin page](/images/portal/1-signin.png)

### Check your email

You'll receive a **Sign in to Playroom** email from `noreply@joinplayroom.com`. Click the **Sign in to Playroom** link inside it.

### Choose a plan

Select a plan. **Hobby** is free.

![Choose a plan](/images/portal/6-choose-plan.png)

### Click + New Project

You'll land on the dashboard. Click the **+ New Project** button in the top right.

![Dev portal dashboard — click New Project](/images/portal/5-dashboard.png)


### Name your project

Enter a project name (letters and numbers only) and click **Continue**.

![Fill in your project name](/images/portal/7-create-project.png)

### Open your project

Your project is created. Click on it in the left sidebar, then click **Configuration**.

![Project overview — click Configuration in the sidebar](/images/portal/8-project-overview.png)

### Generate your Game ID

On the Configuration page, scroll down to the **API key** section. Click **Reset API Key** (or **Create** if this is your first key) to generate your Game ID.

![Generate API Key on Configuration page](/images/portal/10-generate-api-key.png)

### Copy your Game ID

After generating, you'll see your `gameId` is now active in the **General** section. Scroll back to the top to find your `gameId`. You can either:

- Copy just the `gameId` value from the field, **or**
- Copy the ready-made `insertCoin` snippet shown at the top of the page

![Configuration page showing gameId and API key section](/images/portal/11-copy-game-id.png)

</Steps>

## Use the Game ID in your code

Paste the `gameId` into your `insertCoin` call:

```js
insertCoin({ gameId: "your_game_id_here" });
```

<Callout type="info">
  The `gameId` is read-only and assigned when you create a project. You can always find it again in your project's **Configuration → General** tab.
</Callout>
