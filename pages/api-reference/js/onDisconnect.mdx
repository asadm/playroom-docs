# onDisconnect

The callback is also triggered if the player is kicked from the room.

## Signature

```ts
function onDisconnect(callback: (e: { code: number; reason: string; }) => void): void
```

## Parameters

| Name | Type | Description |
|------|------|-------------|
| `callback` | `(e: { code: number; reason: string; }) => void` | Function called with an event containing:  - `code`: number code of the disconnect  - `reason`: human-readable reason |


## Usage

```ts
onDisconnect((event) => {

});
```
